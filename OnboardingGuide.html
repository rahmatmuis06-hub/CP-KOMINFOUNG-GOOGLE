Selamat Datang!

Selamat datang di Dashboard Content Planning Kominfo!

Sistem ini dirancang untuk menggantikan pengelolaan konten manual menggunakan spreadsheet. Dengan dashboard ini, Anda dapat merencanakan, melacak, dan mengelola konten tim secara efisien dan terstruktur.

📝

Menambah Konten

Klik tombol "+ Tambah Konten" di halaman Dashboard untuk membuat perencanaan konten baru.

Isi detail seperti tanggal posting, judul, PIC, platform, dan status konten. Setiap konten akan otomatis tersimpan di database.

📋

Papan Kanban

Gunakan Papan Kanban untuk melihat progress konten secara visual.

Anda dapat drag & drop kartu konten antar kolom (Draft → In Progress → Review → Completed) untuk memperbarui status secara cepat.

📊

Kalender & Statistik

Lihat jadwal posting konten di Kalender untuk memastikan distribusi konten yang merata sepanjang bulan.

Pantau beban kerja tim di halaman Statistik — termasuk distribusi konten per anggota, tingkat penyelesaian, dan peringkat produktivitas.

👥

Peran Tim

Dashboard ini mendukung beberapa peran dalam tim:

  • Kadiv — Supervisi dan persetujuan konten
  • Admin — Input data dan koordinasi tim
  • Graphic Designer — Desain visual (flyer, feeds, story)
  • Videographer — Produksi konten video (reels, TikTok)
